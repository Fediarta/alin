perkalian matriks 3x3.#

  1. Pertama-tama, kita mengimpor pustaka NumPy dengan pernyataan import numpy as np. NumPy adalah pustaka Python yang sangat berguna untuk operasi matriks dan array.

  2. Kemudian, kita mendefinisikan dua matriks 3x3, A dan B, menggunakan fungsi np.array(). Anda dapat mengubah nilai elemen matriks sesuai kebutuhan Anda. Dalam contoh ini, A memiliki nilai [1,2,3],[3,4,5],[9,7,6]B memiliki nilai [4,2,4],[3,7,5],[9,7,7]

  3. Selanjutnya, kita menggunakan fungsi np.dot() untuk melakukan perkalian matriks antara A dan B. Fungsi np.dot() adalah fungsi bawaan NumPy untuk melakukan perkalian titik antara dua matriks.

  4. Hasil perkalian matriks disimpan dalam variabel AB1.

  5. Akhirnya, kita mencetak hasil perkalian matriks menggunakan pernyataan print().

Dengan menggunakan NumPy, kita dapat dengan mudah melakukan operasi matriks seperti perkalian, penjumlahan, dan operasi matriks lainnya dengan cepat dan efisien dalam Python.
